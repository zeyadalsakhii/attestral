# Brand assets

Original marks drawn for Attestral Labs — pure vector geometry, no embedded fonts, no third-party artwork.

- `attestral-labs-seal[-dark].svg` — the org mark: ink fine-tooth notary seal with a six-spoke
  attestation star. Use for Attestral Labs. Ink means the org; seal red is reserved for products.
- `attestral-mark[-dark].svg` — the product mark: seal-red plate, symmetric A with the check as
  its crossbar. Use for Attestral the scanner, favicons, app icons.
- `attestral-labs-avatar.png` — 1024px seal on a paper tile, sized for the GitHub org avatar.
- `wordmark-*.svg` — text lockups; render exactly with Libre Caslon Text (OFL), fall back to Georgia.

The "Attestral" name and these marks identify Attestral Labs and its products. They are **not**
covered by the repository's Apache 2.0 license (see §6: no trademark grant) — don't use them to
brand derivative or third-party products.
